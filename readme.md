# 必要说明

目前项目中很多函数/代码段显得复杂，并且用到了很多处storage，这都是因为后端还没有完成。前端临时模拟了一些后端的功能，比如打卡，获取动态信息、评论，获取存储打卡消息等。

等待后端完成后，会将这些功能迁移到后端。前端代码会调用后端的API来实现这些功能。

# 如何运行？

克隆代码到本地，使用微信开发者工具导入项目。

如果要进行调试，需要在微信开发者工具中打开“不校验合法域名、web-view（业务域名）、TLS版本以及HTTPS证书”选项。

并且小程序需要添加服务器域名 腾讯地图：https://apis.map.qq.com

在 app.js 的globalData中添加：
```
  mapKey: "YOUR_QQ_MAP_KEY", // 腾讯地图API密钥,需要在腾讯地图上注册并且分配配额，并且是反向解析地址的配额
```

如果需要真机调试，需要重新注册一个新的小程序（不是直接扫码登录然后可选的2个测试账号，而是要重新注册，完成主体注册的，才可以上传到微信服务器真机测试）真机测试和模拟器在功能和页面渲染上存在巨大差异！
